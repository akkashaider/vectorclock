1.Program consists of file vector.py
2.To execute program run the following command on console
./vector.py configuration_file 1
3.Example configuration file is included.
4.Configuration file should have ip address or complete hostname of node e.g ukko066.hpc.cs.helsinki.fi
5.It waits for 15 seconds to make sure all other nodes are running before executing algoithm.
6.It randomly selects between local event and send message.
7.It displays messages according to specifications.
